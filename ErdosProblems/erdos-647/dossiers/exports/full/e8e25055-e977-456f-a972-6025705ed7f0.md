Mertens second theorem (∑_{p≤x} 1/p = log log x + M + o(1)) after Abel summation. The
antiderivative of 1/(t·(log t)²) is −1/log t = −(log t)⁻¹, since
  d/dt [−(log t)⁻¹] = (log t)⁻² · (1/t) = 1/(t·(log t)²).
By the FTC, for x ≥ 2,
  ∫_2^x 1/(t·(log t)²) dt = (log 2)⁻¹ − (log x)⁻¹.
Since (log x)⁻¹ > 0 for x ≥ 2, this is bounded above by (log 2)⁻¹. This exact value is the
