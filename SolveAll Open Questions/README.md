# SolveAll Open Questions

Working repo for tackling unsolved problems catalogued at
[SolveAll.org](https://solveall.org/problems) — a community-curated collection of
real, open research problems in the mathematical sciences (statistics, learning
theory, probability, optimization, information theory, and adjacent areas),
maintained as a live benchmark for AI mathematical reasoning.

Snapshot taken 2026-07-17: 120 problems total (99 Open, 19 Partially Resolved,
0 Disputed, 2 Resolved).

Each problem has its own folder under [`problems/`](problems/), numbered by its
position in the site's importance-sorted browse listing. Each folder's `README.md`
carries the metadata scraped from the listing (status, importance, categories,
who posed/sourced it) plus placeholders for the full statement and work notes —
the site sits behind a bot-check that blocks automated fetching, so statements need
to be pasted in by hand from the individual problem pages.
